# Route Optimization (TSP)

This Flutter application demonstrates various solutions to the Travelling Salesperson Problem (TSP) using static and dynamic graph representations. It includes graphical visualizations of the graph nodes and the optimal route.

## Features
- **Graph Representation**: Supports Static (Adjacency Matrix) and Dynamic (Adjacency Linked List) representations.
- **Algorithms**:
  - Brute Force (Exhaustive Search)
  - Greedy (Nearest Neighbor)
  - 2-Opt (Custom Optimization Algorithm)
- **Visualization**: Visually plot cities and routes on a canvas.
- **Performance Metrics**: Computes path cost, execution time, and compares memory representations.

## Tech Stack
- Framework: Flutter
- Language: Dart

## Run Instructions
Ensure you have the Flutter SDK installed.
1. Run `flutter pub get`
2. Run `flutter run`
